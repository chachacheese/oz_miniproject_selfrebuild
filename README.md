# 📊 흡연 여부 데이터 분석 프로젝트
건강검진 데이터를 활용해 **흡연자 vs 비흡연자** 간 건강 지표 차이를 분석하고,  
향후 **흡연 여부 예측 모델 개발을 위한 기초 인사이트**를 도출한 프로젝트입니다.

---

## 📌 프로젝트 개요
해당 프로젝트는 건강검진 데이터를 기반으로 **탐색적 데이터 분석(EDA)** 을 수행하여  
흡연 여부에 따른 주요 건강 지표 차이와 변수 간 관계 구조를 파악하는 것을 목표로 합니다.

> 💡 본 프로젝트는 모델링 이전 단계의 EDA 중심 분석 프로젝트입니다.

---

## 🎯 프로젝트 목표

### 1. 흡연자 vs 비흡연자 건강 지표 차이 분석
- BMI, 혈압, 혈당, 콜레스테롤 등 핵심 지표 비교

### 2. 주요 변수들의 분포 및 상관관계 파악
- 단변량·이변량 시각화를 통한 패턴 탐색  
- 상관계수 기반 변수 간 관계 구조 확인

### 3. 통계적 유의성 검증
- T-test, Mann–Whitney U test를 활용한 집단 차이 검정

### 4. 시각화 기반 건강 인사이트 도출
- 이상치, 분포 차이, 변수 간 관계 구조 중심 분석

### 5. 향후 예측 모델 개발 기반 마련
- 변수 후보 탐색  
- 데이터 불균형 확인  
- 전처리 전략 검토

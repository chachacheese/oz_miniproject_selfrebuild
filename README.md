📊 흡연 여부 데이터 분석 프로젝트

건강검진 데이터를 활용해 흡연자 vs 비흡연자 간의 건강 지표 차이를 분석하고,
향후 흡연 여부 예측 모델 개발을 위한 기초 인사이트를 도출한 프로젝트입니다.

⸻

📌 프로젝트 개요

이 프로젝트는 건강검진 데이터를 기반으로 EDA 중심 분석을 수행하여
흡연 여부에 따른 주요 건강 지표 차이와 변수 간 관계 구조를 탐색합니다.

💡 모델링 전 단계의 데이터 분석(EDA) 프로젝트입니다.

⸻

🎯 프로젝트 목표

✔ 1. 흡연자와 비흡연자 간의 건강 지표 차이 분석

BMI, 혈압, 혈당, 콜레스테롤 등 주요 지표가 집단 간 차이를 보이는지 탐구

✔ 2. 주요 건강변수들의 분포와 상관관계 파악
	•	단변량·이변량 시각화를 통한 패턴 확인
	•	변수 간 상관성 분석(heatmap)

✔ 3. 통계적 유의성 검증
	•	T-test, Mann–Whitney U test 등으로 집단 차이의 통계적 유의성 확인

✔ 4. 시각화를 통한 인사이트 도출
	•	이상치(outlier), 분포 차이, 변수 간 관계 구조 확인

✔ 5. 향후 예측 모델 개발을 위한 기반 마련
	•	변수 후보 도출
	•	데이터 불균형 여부 파악
	•	전처리/변환 전략 검토

⸻

📂 프로젝트 구조 및 파일 설명

✔ 1_1_개인_흡연여부데이터_황진영_수정.ipynb

👉 데이터 구조, 분포 확인, 전처리 단계
	•	데이터 형태 확인(결측치, 자료형, 범주형 여부)
	•	주요 건강 지표의 분포 파악(Histogram, Boxplot 등)
	•	이상치 탐색 및 기초 전처리
	•	변수별 기본 통계량 분석
	•	흡연 여부에 따른 단변량 비교 (EDA Part 1)

⸻

✔ 1_2_데이터분석_황진영_수정완.ipynb

👉 데이터 간 관계 가설 수립 및 검증 단계
	•	변수 간 관계 탐색(Scatterplot, Correlation Heatmap)
	•	흡연 여부 그룹별 지표 차이에 대한 가설 설정
예: “흡연자가 HDL이 낮을 것이다”, “흡연자에서 중성지방이 더 높을 것이다”
	•	T-test, Mann-Whitney U test 등 통계적 검정 수행
	•	시각화 기반 인사이트 도출
	•	향후 모델링을 위한 변수 후보 분석 (EDA Part 2)
